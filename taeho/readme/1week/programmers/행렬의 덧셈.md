## 문제 유형

기본적인 2차원 배열 문제

## 코드 작성 이유

2차원 배열을 2중 반복문으로 모든 요소들에 접근하여 각 위치가 같은 요소들끼리 더
한 후 row 리스트에 담는다. 만들어진 row 리스트를 result 리스트에 담는 과정을 반
복하면 최종 행렬 덧셈의 결과가 나온다. <br>파이썬은 zip 내장 함수를 이용하면 더
간결하게 풀 수 있을 것 같다.
