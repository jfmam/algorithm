# 문제 유형
10진수를 16진수로 변환

# 코드 작성 이유
내장 함수를 이용하여 10진수를 2,8,16으로 변환하는 방법에 대해 알아본다.

# 문제를 못 푼 이유
내장함수를 사용하는 방법을 몰랐다.
## oct,bin,hex
oct,bin,hex 내장함수를 이용하여 10진수를 변환 할 수 있다.
## format
format(10진수, 'b,o,x')를 이용하여 변환 된 수를 출력할 수있고 #을 붙이면 접두사를 붙여서 출력한다.
0b: 2
0o: 8
0x: 16
