# 코드 수정 사항
## set함수 사용
set함수를 사용하게 되면 다음과 같이 3단계를 거친다.
1. 변수를 set 선언 arr = set()
2. add 함수를 이용하여 정수를 arr에 append 시켜준다. set함수일 때에는 add를 사용한다.
3. iterable형식임으로 list(arr) 를 이용하여 다시 배열화 시킨다.
